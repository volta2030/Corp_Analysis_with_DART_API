# Corp_Analysis_with_DART_API(CADA)
DART Open Api를 이용하여 기업의 EPS, PBR 등을 계산하는 프로그램입니다.

## How to activate

### 1. 필요 라이브러리 설치

```
pip install requirements.txt
```

### 2. DART Openapi에서 인증키 신청

https://opendart.fss.or.kr/

### 3. api_key.txt에 인증키 입력 후 실행
```
python main.py
```
